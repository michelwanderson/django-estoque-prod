1. Criando o projeto Django
django-admin startproject core
cd core

2. Testa:
python manage.py runserver
http://127.0.0.1:8000

3. Criando os apps do sistema - Vamos criar os dois módulos principais:
python manage.py startapp estoque
python manage.py startapp producao

4. Criar usuario admin
python manage.py createsuperuser

4. Atualizar alteraçoes no banco
python manage.py makemigrations
python manage.py migrate

6. apagar o banco
rm db.sqlite3
rm estoque/migrations/0*.py
